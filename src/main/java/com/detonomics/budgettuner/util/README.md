# Util Package

This package contains utility classes and helper functions used throughout the application.
Subpackages:
- `ingestion`: specific utilities for data ingestion.

Key classes include:
- `BudgetFormatter`: Formats budget amounts for display.
- `GuiUtils`: Helper methods for GUI navigation and chart setup.
- `LogarithmicAxis`: Custom JavaFX chart axis for logarithmic scaling.
- `DatabaseManager`: Manages database connections and initialization.
- `PlotlyHelper`: Utility for integrating Plotly charts (if used).
