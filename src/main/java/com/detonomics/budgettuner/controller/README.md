# Controller Package

This package contains the main application controllers and entry points.
Key classes include:
- `GuiApp`: The main entry point for the JavaFX GUI application.
- `Launcher`: Helper class to launch the GUI application.
- `BudgetTunerCLI`: The main entry point for the Command Line Interface.
- `WelcomeController`: IT Handles the main welcome screen logic.
- `BudgetController`: Handles the budget viewing and selection logic.
- `BudgetDetailsController`: Manages the detailed view of a specific budget year.
- `AnalysisController`: Controls the analysis view (charts and breakdown).
- `BudgetComparisonController`: Handles the logic for comparing two budget years.
- `ComparisonDetailsController`: Manages the detailed comparison view.
- `IngestController`: Controls the UI for the budget ingestion process.
